# Vending Machine Using Verilog

This project implements a simple **Vending Machine** using **Verilog HDL** based on a **Finite State Machine (FSM)**. It allows users to select a product, enter the amount, and calculates the remaining balance.

## Features
- FSM-based design
- Four product options: Chips, Soda, Coke, Chocolate
- Balance calculation
- Reset functionality
- Manual and random item selection
- Verilog testbench for simulation

## Tools Used
- Verilog HDL
- Digital Logic Design
- FPGA/Simulation Tools

## Files
- `vend_mac.v` – Verilog source code
- `vend_mac_tb.v` – Testbench
- `Mini_Project_Report.pdf` – Project report
